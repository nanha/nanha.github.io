6. 문제 해결

6.1 PATH 문제

VS Code에서 npm/node를 찾을 수 없는 경우:

  • 전체 경로 사용: /Users/username/.nvm/versions/node/v24.12.0/bin/node
  • bash shell에서 nvm 로드: source ~/.nvm/nvm.sh

6.2 빌드 오류

TypeScript 컴파일 오류가 발생하면:

  • inputSchema가 올바른 Zod 스키마인지 확인
  • tsconfig.json 설정 확인
  • 의존성 재설치: rm -rf node_modules && npm install

6.3 서버 연결 실패

MCP 클라이언트에서 서버에 연결할 수 없는 경우:

  • 서버가 실행 중인지 확인
  • 경로가 올바른지 확인
  • 로그 확인: console.error() 메시지
